API Documentation for "owners" Model

Create Owner

Create a new owner.

HTTP Method: POST

URL: `/create/`

Python Example:

                    
import requests

url = 'https://your-api-url.com/create/'
data = {
    'username': 'newuser',
    'email': 'newuser@example.com',
    'password': 'password123'
}

response = requests.post(url, data=data)
print(response.status_code)

JavaScript Example:

                    
fetch('https://your-api-url.com/create/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        username: 'newuser',
        email: 'newuser@example.com',
        password: 'password123'
    })
})
.then(response => {
    console.log(response.status);
});

Update Username and Phone

Update the username and phone number of an owner by ID.

HTTP Method: PUT

URL: `/update/up/{id}/`

Update Email

Update the email of an owner by ID.

HTTP Method: PUT

URL: `/update/email/{id}/`

Delete Owner

Delete an owner by email.

HTTP Method: DELETE

URL: `/delete/{email}/`

Owner Details

Retrieve details of an owner by username.

HTTP Method: GET

URL: `/detail/{username}/`

Update Password

Update the password of the authenticated owner.

HTTP Method: PUT

URL: `/update_password/`

Delete My Account

Delete the authenticated owner's account.

HTTP Method: POST

URL: `/deleteme/`

Verify OTP

Verify the OTP code sent to the owner's email.

HTTP Method: POST

URL: `/verify/`

Verify Email

Resend the email verification OTP to the authenticated owner's email.

HTTP Method: GET

URL: `/verify-email/`

Owners List

Retrieve a list of all owners.

HTTP Method: GET

URL: `/`

My Profile

Retrieve the profile details of the authenticated owner.

HTTP Method: GET

URL: `/profile/`

New Email

Change the email of the authenticated owner and request verification.

HTTP Method: POST

URL: `/new-email/`
